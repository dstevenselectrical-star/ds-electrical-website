What We Install

Five disciplines.
One contractor.

A working network is five jobs done right and joined up: the cabling, the cabinet, the switching, the wireless, and the fibre back-haul. We do all five — in-house, with the same crew, to the same standard.

01 · Wireless
📶

Wi-Fi Access Points

Site-wide commercial Wi-Fi distribution. Reyee & Ruijie PoE access points, indoor and outdoor (IP67), single SSID with seamless roaming, free cloud controller.

  • Schools, surgeries, libraries, factories
  • Indoor + outdoor IP67 variants
  • Reyee Mesh, single-SSID roaming
Wi-Fi page →
02 · Switching & Power
⚡

PoE Networking

Managed PoE+ and PoE++ switches sized to power the whole site. Access points, IP cameras, IP phones, access control, kiosks — one cable per device, no local power supplies needed.

  • PoE+ (30 W) and PoE++ (60-90 W)
  • Power budget calculated, not guessed
  • VLAN-capable for staff / guest / IoT
PoE page →
03 · Cabinets
🔐

Network Cabinets & Patch Panels

Wall-mount and floor-standing cabinets, patch panels, faceplates, dressed and labelled to a cable test sheet. Power, cooling and cable management done properly — not a heap of spaghetti behind a door.

  • 6U wall-mount to 42U floor-standing
  • Patched, dressed, labelled both ends
  • PDU, cooling, lockable doors
Cabinets page →
04 · Cabling
🔗

Structured Data Cabling

Cat6 and Cat6a structured cabling installed in containment, terminated to TIA-568, cable tester certified to permanent-link limits. PDF test report per port at handover.

  • Cat6 (1 Gbps) / Cat6a (10 Gbps)
  • LSZH-sheathed for commercial
  • tested, fully labelled
Cabling page →
05 · Backbones
✨

Fibre Optic Backbones

Inter-building links, long runs, noise-critical environments. City & Guilds Level 3 fibre qualified, Fujikura fusion splice, OTDR test, channel test report.

  • OM3 / OM4 / OM5 multimode
  • OS1 / OS2 singlemode for campus
  • Fusion splice + OTDR test
Fibre page →
06 · Network-Connected
📷

IP CCTV (over your network)

4K IP cameras, NVRs and PoE-powered turrets / bullets / PTZ that ride on the same Cat6a, PoE switching and cabinets we install for the network — one job, one cable plant.

  • 4K IP cameras, NVRs, remote view
  • Same PoE switch, same cable plant
  • VLAN-isolated from staff network
