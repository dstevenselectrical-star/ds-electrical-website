How it works

Five steps from survey to grant claim.

From first phone call to handover with the OZEV grant claimed — the process we run for every commercial EV install, from a single workplace charger to a 40-socket bay.

Site survey

45–90 minutes on site. Supply phase + capacity check, cable-route mapping, civils assessment, dwell-time interview, brand selection against use case. Existing fuse board photographed.

DNO notification + load assessment

Live load profile measured at the supply meter. ENA EV connection form (or G99/G100) submitted to the DNO. DLM topology designed where 4+ chargers exceed available capacity.

Fixed quote + OZEV claim

One price per scope, broken down per socket. Materials, civils, hardware, certification all included. OZEV voucher application submitted on your behalf; voucher deducted at install stage.

Install + commission

First-fix cabling and civils, sub-board installed, chargers mounted and commissioned to manufacturer spec. DLM controller paired and tested live. Tariff and authorisation configured.

Handover + OZEV claim

EIC per circuit, NAPIT compliance cert from the national database, DNO confirmation, OZEV grant deducted from the invoice and claim closed out. PDF pack emailed; warranty registered.

FAQ

Common commercial EV questions.

How does load balancing work on a multi-charger fleet install?

Dynamic load management (DLM) sits between the supply meter and the chargers. It measures the building's total real-time draw and dynamically reduces output across the connected EVs so the incoming supply is never overloaded. On a 4-charger install with a 100 A three-phase supply, that means all four cars can plug in simultaneously and the system tunes each socket up or down — overnight when the kitchen and lighting load drops, every car gets full 22 kW; at lunchtime when peak load is on, output is throttled to keep the main fuse safe. It removes the need to upgrade the DNO supply for most multi-charger sites.

How does the OZEV Workplace Charging Scheme grant work?

The OZEV Workplace Charging Scheme covers up to £350 per charge-point socket, capped at 40 sockets per applicant company, against the cost of installation. The business applies online before the install begins and receives a voucher code valid for 6 months. The voucher is given to us at the install stage and we redeem it directly with OZEV — the grant is deducted from your invoice rather than refunded after the fact. Eligibility: registered UK business, charity or public-sector body with off-street parking dedicated to staff or fleet use. We handle the full claim paperwork as part of the install.

How long does DNO notification take and do I need to wait for it?

For single 7 kW chargers most DNOs only require notification (no approval) and we file it on the day. For 22 kW chargers and any multi-charger install where the combined demand exceeds 60 A per phase or the supply is upgraded, the DNO requires prior approval — typically 10–15 working days, occasionally longer if a supply upgrade is needed. We submit the notification at quote stage so DNO sign-off lands before the install date. We handle the full process — site EV-readiness assessment, ENA G99 / G100 / EV connection form as applicable — included in the quote.

Can you install on a leased commercial property?

Yes — most of our hotel, restaurant and workplace installs are on leased premises. You'll need landlord consent before the install and ideally a written agreement on what happens to the equipment at end of lease (stays with the building or is removed and made good). We're happy to write to your landlord with the proposed install scope, cable routes, and confirmation that the work is fully reversible if required. The OZEV grant is also valid on leased premises provided the lease has 18+ months remaining.

Why Type A RCBO and not Type B on EV chargers?

BS 7671 18th edition (Amd 2) requires every EV charge point to have residual current protection that detects DC fault currents. Modern Hypervolt, Zappi, Ohme, Andersen, Project EV and Wallbox commercial units all include integrated 6 mA DC fault detection on board, which means the upstream protective device only needs to be a Type A RCBO — not the much more expensive Type B. Specifying Type B on these chargers is a common over-specification that costs hundreds of pounds per socket for no compliance benefit. Every charger we install ships with the integrated DC monitoring, so every one of our installs uses a Type A 30 mA RCBO at the supply board.

Do you install DC rapid chargers (50 kW+)?

No — DC rapid chargers (50 kW and above) require an HV-trained team, a dedicated transformer and a substantial DNO supply upgrade, and we don't claim that as a core service. We focus on AC charging up to 22 kW per socket, which covers virtually every workplace, hotel destination and fleet-overnight use case. If you genuinely need rapid (typically only fleets where vehicles cycle through one charger multiple times per day with no overnight dwell), we partner with specialist rapid-charging contractors and can scope and project-manage the AC + rapid combination — but the rapid hardware install itself goes to the partner, not us.
